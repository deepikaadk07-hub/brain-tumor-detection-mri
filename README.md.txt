# Brain Tumor Detection using MRI

## Description
This project detects brain tumors from MRI scans using deep learning (U-Net).

##  Dataset
BraTS 2020 MRI dataset (.nii format)

Dataset is not uploaded due to large size.

Download from:
https://www.kaggle.com/datasets/awsaf49/brats20-dataset-training-validation

##  Features
- MRI Visualization
- Tumor Segmentation
- 3D Tumor Analysis
- Severity Classification

## How to Run

pip install -r requirements.txt

python src/train.py

python src/predict.py

## Output
- Tumor mask
- Dice Score
- IoU Score
- Tumor volume
